Mereb Jenkins Helper

Mereb Jenkins Helper works around one project unit: a Mereb config file plus its sibling Jenkinsfile. The plugin auto-detects that pair for the current project.

Preferred config path: .ci/ci.mjc
Legacy files .ci/ci.yml and ci.yml still work and can be migrated.

How the tool window works

  • Open the Mereb Jenkins tool window from the right-side tool window bar.
  • If the workspace contains one Mereb project, the tool window focuses it automatically.
  • If the workspace contains multiple Mereb projects, use the Project selector to choose one.
  • The tool window refreshes automatically when the focused config or sibling Jenkinsfile changes.
  • The Jenkins tab polls live Jenkins data while the tool window is visible and the project is mapped.

Connect to Jenkins

  • Use Connect Jenkins in the tool window or open Settings / Preferences -> Tools -> Mereb Jenkins Helper.
  • Enter the Jenkins base URL, your Jenkins username, and an API token.
  • Use Open Token Page to jump to /me/configure in Jenkins and create a token.
  • The plugin stores the token in IntelliJ Password Safe, not in the repo or in .ci/ci.mjc.

How job mapping works

  • The plugin searches visible Jenkins jobs for likely matches based on the focused project root.
  • If one exact match exists, it is selected automatically.
  • If several jobs are plausible, the plugin asks you to choose one and remembers that mapping locally.
  • If the saved job later disappears, the plugin asks you to remap it.

What to do in the editor

  • Use completion while typing for known keys and values such as recipe, delivery.mode, and release.autoTag.bump.
  • Hover important keys to see Mereb-specific runtime notes and defaults.
  • Use quick fixes when the plugin flags issues like missing recipe, invalid environment order, or legacy config paths.
  • Look for inline block descriptions above important sections such as deploy, image, and release.

Useful actions

  • Tools -> New Mereb Jenkins Config creates a starter .ci/ci.mjc.
  • Tools -> Migrate Mereb Jenkins Config previews a conservative migration from legacy filenames or older config shapes.
  • Tools -> How To Use Mereb Jenkins Helper reopens this guide at any time.

How to read the tool window

  • Overview: resolved recipe, enabled capabilities, notices, and safe fixes.
  • Jenkins: connection state, mapped job, recent runs, stage status, pending input, and artifact links.
  • Flow: the derived runtime sequence the current config implies.
  • Relations: links between order lists and environment blocks, plus unused or missing definitions.
  • Upstream: manual schema freshness check against the current GitHub schema.
